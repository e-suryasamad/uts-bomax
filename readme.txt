BMI - Body Mass Index App

QR Code = Login (read API)
API = username&password, identity, bmi standard, result count of bmi (with date when count it)
C-R-U-D = create account, count of bmi, - read idendity, bmi standard, result count of bmi, - update indentity, - delete result count of bmi
Navigasi = home, registrasi, login, form input, bmi standard, result count of bmi

menu di drawer(screen)->alur stack navigasi(screen) berikutnya(jika ada): 
1. home(countBMI)->result count of bmi (jika login dan disimpan), 
2. Login->count bmi, 
3. Registrasi->count bmi, 
4. BMI standard, 
5. Identity/Account->update identity, result count of bmi->delete result count of bmi
